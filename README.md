# jitterbit_java_cleanarch

Desafio de teste teorico da JitterBit, no teste é solicitado para que seja feito em Node.JS, mas, para fins de estudo realizei em Java 25 + SpringBoot, usando PostgreSql como banco de dados, autenticação com JWT usando SpringSecurity.
